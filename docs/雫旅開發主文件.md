## 第一部分：已完成的除錯修正（請 Cursor 先確認）

### 修正 1：adminGetAllOrders() 強制序列化（main.js）

**根本原因：**
google.script.run 傳回前端時，若資料含非標準型別
（空字串混入數字欄位如 extraBeds/paidDeposit、boolean 如 reminderSent），
GAS 序列化失敗直接回傳 null，導致前端 Array.isArray(data) === false，
allOrders 永遠是空陣列，表格顯示「共 0 筆」。

**修正後的 adminGetAllOrders()（main.js）必須是：**
```javascript
function adminGetAllOrders() {
  var orders = DataStore.getOrders();
  return JSON.parse(JSON.stringify(orders)); // 強制清理，避免 GAS 傳 null
}
```
請確認此函式有 JSON.parse(JSON.stringify(orders)) 這行。

### 修正 2：後台所有操作必須用 google.script.run，不能用 fetch（admin.html）

**根本原因：**
admin.html 跑在 googleusercontent.com iframe 內，
用 fetch POST 打 script.google.com/exec 會被 CORS 擋。

**必須用 google.script.run 的函式清單：**
- loadOrders() → adminGetAllOrders
- saveOrder() → adminUpdateOrderAndSync
- submitAddOrder() → adminCreateBooking
- sendEmailNotification() → adminSendNotificationEmail
- showLineNotification() → adminGenerateNotification
- getCalendarStats() → adminGetCalendarStats
- rebuildCalendars() → adminRebuildCalendars
- clearCalendars() → adminClearCalendars
- cleanupOldYear() → adminCleanupOldYear

（loadOrders 正確版本見主文件內文）
